## Install Extension

```bash
  git clone  https://github.com/vishal2376/cf-solutions
  cd cf-solutions
```

- After cloning, goto Extension settings in your Chrome browser.
- Enable `developer mode`.
- Click on `load packed` and open `cf-solutions` folder.
